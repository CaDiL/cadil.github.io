Ubuntu контейнер ажилууллах docker run ubuntu Ubuntu контейнер руу shell -ээр хандах docker run -ti <your-container-name> /bin/bash Эсвэл: docker exec -ti <your-container-name> bash Note: shell-с гарахдаа Ctrl + D Ажиллаж байгаа контейнер инспект хийх (inspecting) Контейнер нэрийг тодорхойлох. docker ps -a Контейнер нэрийг ашиглан инспект хийх. docker inspect <your-container-name> IP хаяг тодорхойлох docker inspect <your-container-name> | grep IPAddress docker inspect --format {{.NetworkSettings.IPAddress}} <your-container-name> Лог шалгах docker logs <your-container-name> Контейнер зогсоох docker stop <your-container-name> Ажиллаагүй байгаа контейнер цэвэрлэх (clean up) docker rm -v $(docker ps -aq -f states=exited)

Гипервизор (Виртуал машин мониторинг ВММ) Виртуал машин бүрийн нѳѳцийг тооцоолж динамик хуваарилалт хийнэ. Контейнр: Хѳнгѳн байх бүтэцтэй виртуалчлал (контейнэр) нэг хостинг дотор ажиллах боломж. Үйлдлийн системээс ялгаатай нь кернел байхгүй. Хостинг хийж байгаа машины кернелийг хуваалцаж ашиглана (Илүү хѳнгѳн бүцэтэй болгох гэж тэр).
